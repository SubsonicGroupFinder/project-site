How you can contribute

The project is open to anyone who can help. We have many roles for all kinds of contributors from developers fluent in react js, Django Python and MY SQL databases as well as tons of other roles such as graphics designers. Additionally in the future we will require testers and package maintainers to name a few roles.

To find out if you can help or if you want to offer your services please check our github repository or contact in the linked below project discord server.

If you are interested in the project then please check out our how to contribute page
